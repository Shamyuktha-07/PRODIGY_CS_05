A lightweight network analysis tool built in Python to intercept and inspect live traffic.

## Features
- **Real-time Capture:** Monitor IPv4 packets.
- **Protocol Analysis:** Distinguish between TCP, UDP, and ICMP.
- **Payload Inspection:** View raw data to understand encryption (TLS/SSL).

## Educational Purpose
This tool was developed to visualize the OSI model in action and demonstrate why encryption (HTTPS/TLS) is vital for data privacy.
